
# Introdução
Scyther Maker é um software baseado no metasploit-framework e utiliza muitas das funcionalidades ja existentes nele.

para automação de payload de msfvenom/msfconsole

com alvo em celulares android.

objetivo: tornar o teste de segurança a dispositivos moveis mais acessivel e prático.

principal funcionalidade:

a troca automatica da versão sdk para a correta.

# Instalação e configuração
execute o setup para instalar as dependencias: bash setup.bash

inicie o scyther maker[sm.bash] bash sm.bash 

alternativa silenciosa(Quiet Mode): bash sm_iniciar.bash
